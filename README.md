# AI-Based Student Performance Prediction System

## Project Overview
This project predicts the final performance (G3) of students using Machine Learning.

## Features
- Data Preprocessing
- Data Visualization
- Feature Selection
- Train-Test Split
- Random Forest Regression
- Model Evaluation
- Student Performance Prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset
Student Performance Dataset (student-por.csv)

## Machine Learning Algorithm
Random Forest Regressor

## Results
- MAE: 0.798
- R² Score: 0.818
